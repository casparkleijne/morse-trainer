# Morse Code Trainer

A retro-styled morse code trainer using the Koch method, built with Godot 4.5.

## Features

- **Koch Method**: Progressive learning starting with K and M, adding characters at 90% accuracy
- **Sliding Window Accuracy**: Mastery based on last 20 attempts per character
- **Configurable Audio**: Speed (15-60 WPM) and frequency (440-1320 Hz)
- **Statistics**: Per-character mastery tracking
- **Retro Aesthetic**: 8-bit style with ANSI color palette

## Koch Sequence

`K M R S U A P T L O W I . N J E F 0 Y V G 5 / Q 9 Z H 3 8 B ? 4 2 7 C 1 D 6 X`

## Getting Started

```bash
git clone https://github.com/casparkleijne/morse-trainer.git
```

Open in Godot 4.5+ and run with F5.
